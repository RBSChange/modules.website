Comment éditer le contenu d'une page?

Pour éditer le contenu d’une page (c’est-à-dire visualiser le rendu de votre page), deux possibilités :

  • Ou bien cliquer sur le titre de la page, puis sur l'icône dans le menu de navigation du haut de l’applicatif
  • Ou bien effectuer un cliquer-droit sur le titre d’une page (au niveau de la fenêtre du centre de l'écran), puis cliquer sur «Editer le contenu de la page» :

La fenêtre Editeur du contenu s’affiche alors :

La page se charge avec la représentation des éléments en durs du site, les blocs non modifiables sur la page (bandeau du haut de page, menu de navigation gauche, menu du bas de page).

D’autres blocs délimités par des pointillés et hachurés sont présents selon le gabarit choisi. Ceux-ci permettent d’insérer du contenu dans la page.

Un menu spécifique à la gestion de l’affichage des pages apparaît :

  • Pour revenir à la liste des pages de la rubrique préalablement sélectionnée, cliquer sur
  • Pour enregistrer les modifications effectuées sur la page, cliquer sur
  • Pour enregistrer cette page comme gabarit (et servir à créer d’autres pages sur le même modèle, cliquer sur . La page sauvegardée en tant que gabarit s'integrera et s’affichera alors dans la liste des gabarits dynamiques lors de la création des pages.
  • Pour pré-visualiser la page dans le contexte du site, cliquer sur

Une fois le bloc hachuré sélectionné, il es possible de :

  • Cliquer sur pour supprimer un bloc préalablement inséré dans la page
  • Cliquer sur pour donner une largeur fixe en pixels à un bloc (à renseigner tel que ‘100px’ par exemple)
  • Cliquer sur pour donner une hauteur fixe aux blocs vides uniquement sur la page (à renseigner tel que ‘100px’ par exemple)
  • Cliquer sur pour réactualiser le contenu de la page
