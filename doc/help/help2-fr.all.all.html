L'insertion du contenu

La partie modifiable de la page peut recevoir différents blocs de contenus disponibles en bas de la page :

Deux blocs sont généralement utilisées :

  • le bloc de texte (pour recevoir du texte mais aussi des images)
  • le bloc vide (qui crée de l’espace pour aérer la page).

Pour insérer un bloc sur la page:

  • Choisir le bloc dans la liste ("Vide" ou "Texte")
  • Le glisser-déposer dans la fenêtre d’édition de la page. Ce bloc est à placer par rapport au bloc hachuré de départ pour le positionner. En effet, on peut placer un bloc uniquement par rapport à un bloc de départ.

Une ligne verte apparaît afin de visualiser l’endroit où le bloc se placera exactement.

Dès lors, insérer son contenu après avoir positionné le curseur au sein du bloc en question (voir également la partie "Les spécificités du bloc de texte").

Pour supprimer un bloc :

  • Le sélectionner
  • Positionner le curseur dans ce bloc afin qu’un liseré rouge l’entoure
  • Cliquer sur

Une confirmation de suppression est alors demandée :

  • Cliquer sur pour confirmer la suppression ou sur pour l’annuler.
